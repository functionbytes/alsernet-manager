# Plan Agent - Quick Start Guide (5 min)

**Get started with Plan Agent in 5 minutes.**

---

## What is Plan Agent?

The **Plan Agent** helps you design implementation strategies BEFORE you write code. It breaks down complex features into manageable tasks and creates clear development roadmaps.

Plan [feature name] with:

---

## 5 Real Examples

### Example 1: Simple Feature
```
Plan a customer review system where:
- Users can submit reviews (1-5 stars)
- Reviews are displayed on product page
- Admin can delete inappropriate reviews
- Email notification to product owner
```

**Time to plan:** 5 minutes
**Output:** Task list, database schema, API endpoints

---

### Example 2: Medium Complexity
```
Plan a wishlist feature for customers:
- Users can add/remove products from wishlist
- View wishlist with product details
- Share wishlist with friends (link)
- Get notifications when wishlisted items are on sale
- Admin can see wishlist trends
```

**Time to plan:** 10 minutes
**Output:** Requirements, schema, task breakdown, risks

---

### Example 3: Complex Feature
```
Plan a returns management system:
- Customers request return (with reason)
- Warehouse validates return
- Generate return label
- Process refund
- Track return status in real-time
- Analytics for return trends
```

**Time to plan:** 15 minutes
**Output:** Complete roadmap, schema, risks, deployment plan

## 3-Step Process

### Step 1: Request Plan
```
Ask: "Plan [feature] with [requirements]"
Time: Instant
```

### Step 2: Review Plan
```
Read through the output
Identify clarifications needed
Mark any disagreements
Time: 5-15 minutes
```

### Step 3: Use Plan with Frontend/Backend Agents
```
Now use Frontend Agent for UI
Now use Backend Agent for API
Follow the task breakdown from Plan
Time: Implementation time
```
